SeLoger — Bookmarklet (le plus rapide)

Installation (1 seule fois)

Glisse le bouton ci-dessous dans ta barre de favoris :

📥 Scrape SL

Utilisation

  1. Va sur SeLoger Paris
  2. Clique le bookmarklet 📥 Scrape SL dans tes favoris
  3. Un compteur apparait en haut de page
  4. Change de page (?page=2, ?page=3...)
  5. Re-clique le bookmarklet sur chaque nouvelle page
  6. Quand fini, clique COPIER sur le compteur
  7. Colle dans le dashboard + SeLoger

Astuce rapide

Change juste le numero dans l'URL : ?page=2 → ?page=3 → etc. et clique le bookmarklet a chaque fois. ~5 sec par page.
